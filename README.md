# Simple Python Project

This small Python project includes:

- A `simple_project` package under `src/`
- A calculator module with basic operations
- A CLI entrypoint via `python -m simple_project`
- Unit tests under `tests/`

## Install

```bash
python -m pip install -e .
```

## Run

```bash
python -m simple_project add 3 5
```

## Test

```bash
python -m unittest discover -s tests
```
